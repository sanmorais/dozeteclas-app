# 🎹 Doze Teclas | Laboratório Musical

![Status](https://img.shields.io/badge/status-est%C3%A1vel-success)

O **Doze Teclas** é um ecossistema digital desenvolvido para simplificar o aprendizado da teoria musical e harmonia. Unindo princípios de design interativo, tecnologia web e engenharia de processos, o projeto oferece uma abordagem visual e direta ao ponto para músicos.

---

## 🚀 O Projeto

Diferente dos métodos tradicionais que focam em memorização abstrata, o Doze Teclas transforma conceitos complexos em ferramentas interativas. O projeto nasceu da necessidade de referências rápidas em todos os 12 tons, eliminando a barreira da transposição mental imediata para quem ainda está construindo sua base musical.

### Módulos Atuais:
- **Ciclo de Quartas e Quintas:** Prática interativa de memorização das sequências harmônicas.
- **Relativas Menores e Maiores:** Visualização rápida das conexões entre tonalidades.
- **Campo Harmônico:** Estrutura completa de acordes para suporte à composição e execução.
- **Dashboard Minimalista:** Interface focada no usuário, otimizada para dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído priorizando a performance e a acessibilidade, utilizando tecnologias nativas da Web:

- **HTML5 & CSS3:** Estruturação semântica e interface responsiva (Mobile First).
- **JavaScript (Vanilla):** Lógica de teoria musical, manipulação de DOM e algoritmos de validação.
- **Web Audio API:** Geração de síntese sonora em tempo real para feedback auditivo imediato.
- **Canvas API:** Renderização gráfica dos ciclos e elementos visuais dinâmicos.

---

## 📐 Diferenciais de Design

Como um projeto desenvolvido por um **Publicitário e Músico**, o Doze Teclas segue três pilares:
1. **Kaizen:** Melhoria contínua da interface baseada na experiência do usuário.
2. **Visualização Direta:** Referências prontas em todas as tonalidades (sem a limitação do "Dó Maior").
3. **Foco no Essencial:** Eliminação de "ruídos" pedagógicos para acelerar o raciocínio musical.
